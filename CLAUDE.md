# Claude Code Instructions

This is a dotfiles repository for managing personal configurations across Linux and Windows.

## Structure

- `linux/` - Arch Linux configs
- `windows/` - Windows configs (Git Bash)
- `docs/` - Setup documentation
- `sync.sh` - Sync script (diff/apply/save)
